# **App Name**: 3 AM Zoomies

## Core Features:

- Front Page Song Preview: Feature a single song on the front page for all visitors to preview.
- Secure Payment Processing with Stripe Webhooks: Integrate Stripe webhooks to handle secure transactions for album purchases via Stripe payment links.
- Artist Information: Provide a dedicated section with detailed information about the artist.
- Complete Album Access for Paying Customers: Grant access to the rest of the songs on the album for customers who have completed a purchase.
- Song Lyrics: Display song lyrics

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey creativity and sophistication.
- Background color: Dark gray (#212121) for a modern, immersive experience.
- Accent color: Bright pink (#E91E63) for call-to-action buttons and highlights.
- Headline font: 'Belleza' (sans-serif) for headings, capturing an artistic and stylish feel; Body font: 'Alegreya' (serif) for readability and a touch of elegance.
- Use clean, minimalist icons to represent different music genres and functions.
- Design a grid-based layout with clear visual hierarchy for easy navigation and content discovery.
- Incorporate subtle animations and transitions to enhance user engagement and create a dynamic experience.
- The site should have a minimalist header with the following links: COLLECTIONS, MERCH, MUSIC, FILM, BACK IN STOCK, SHOP INTERNATIONAL
- In the top right corner, the following icons are present: email, search, user, cart
- Album covers displayed like the XO store, featuring the album art and a vinyl record mockup.
